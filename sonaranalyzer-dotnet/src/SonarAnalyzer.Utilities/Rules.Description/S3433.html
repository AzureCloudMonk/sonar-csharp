Whether or not they have a test attribute, non-public methods are not recognized as tests, and therefore not executed.

Also, In general, generic and async void test methods are not executed by test frameworks, therefore it's better to avoid them altogether.

A method is detected as test method if marked with one of the following attributes [TestMethod] (for mstest), [Fact] (for xunit) or [Test] (for nunit).

Noncompliant Code Example

[TestMethod]
void TestNullArg()  // Noncompliant
{  /* ... */  }

[TestMethod]
public async void MyIgnoredTestMethod()  // Noncompliant
{ /* ... */ }

[TestMethod]
public void MyIgnoredGenericTestMethod<T>()  // Noncompliant
{ /* ... */ }

Compliant Solution

[TestMethod]
public void TestNullArg()
{  /* ... */  }
